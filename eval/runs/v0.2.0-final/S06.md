## Vocabulary: "cliente" / "clientes" — two entities, one word

This workspace contains two distinct entities that Portuguese speakers both call "cliente":

1. **Shopify buyer** — a person who placed orders in the connected Shopify store. This is what store owners almost always mean.
2. **Organization of type=client** — a business/account record in the platform itself (assumed: the org entity is distinguished by `type=client` — flag if the field differs).

**Resolution rule:** when the question involves purchases, orders, revenue, recurrence, or store performance — signals like "compraram", "pedidos", "recorrentes", "faturamento", "este mês", "ticket médio" — "clientes" means Shopify buyers. Read it as the organization entity only when the question is explicitly about platform accounts or contracts ("clientes cadastrados na plataforma", "organizações", "contas"). If neither signal is present, default to Shopify buyers and state that interpretation in one sentence, offering the other reading.

**"Clientes recorrentes"** = repeat buyers: Shopify customers with an order in the asked period AND at least one earlier order (assumed definition — flag if the business counts 2+ orders inside the period instead).

**Worked example:**
- User: "quantos clientes recorrentes tivemos este mês?"
- Correct: query Shopify customer/order data for the current calendar month; count customers with an order this month who also have at least one prior order. Answer: "Tivemos 37 clientes recorrentes em julho — compradores com pedido este mês e pelo menos um pedido anterior."
- Incorrect: querying organizations of type=client and answering "no clients found".

**Missing/empty data:** never answer "no clients found". If the Shopify data source is unavailable, say so and name the source. If the query succeeds with zero results, report zero explicitly with the period and definition used ("0 clientes recorrentes entre 1–23 de julho, contando compradores com pedido no mês e um pedido anterior").

**Answer format:** reply in the user's language; lead with the number, then one short sentence naming the period and the definition applied. Mention the buyer-vs-organization distinction only when the question was genuinely ambiguous.
